# User Flows for https://www.activeandfitdirect.com/

## 1. Homepage Visit
- User navigates to https://www.activeandfitdirect.com/
- Page loads with title "Active&Fit Direct"
- Main heading "Flexible, Affordable" is displayed
- Body content is visible

## 2. Gym Search
- User visits homepage
- Locates search bar with placeholder "Find a gym near you"
- Enters location (e.g., "New York")
- Initiates search (functionality may require additional setup)

## 3. Check Eligibility
- User visits homepage
- Dismisses any modal by clicking "CONFIRM" if present
- Clicks "Check Eligibility" link
- Navigates to eligibility page (URL contains /eligibility)

## 4. Find Your Gym
- User visits homepage
- Dismisses any modal by clicking "CONFIRM" if present
- Clicks "Find your gym" link
- Navigates to search page (URL contains /search)

## 5. Access Workout Videos
- User visits homepage
- Clicks "Workout Videos" link
- Navigates to workout videos section

## 6. Explore Exclusive Features
- User visits homepage
- Views "Discover Exclusive Features" section
- Sees Marketplace and Workout Videos features

## 7. Browse Marketplace
- User visits homepage
- Views "Unlock Top Brands for Less" section
- Explores marketplace offerings

## 8. Footer Navigation
- User visits homepage
- Scrolls to footer
- Clicks on "About Us", "FAQ", or "Contact Us" links
- Navigates to respective pages

## 9. Mobile Responsiveness
- User accesses site on mobile device (viewport 375x667)
- Homepage elements remain visible and functional
- User types 92101 into the search textfield
- User clicks the Enter key
- User is Navigated to the Search page with 92101 visible on the page

## 10. Analytics - Google Analytics calls
- User navigates to https://www.activeandfitdirect.com/search
- Page loads and any modals are dismissed if present
- The page makes network requests to Google Analytics endpoints (for example `google-analytics.com`, `analytics.google.com`, `googletagmanager.com`)
- Verify that at least one analytics/measurement request is sent (e.g., `/collect`, `/g/collect`, or `gtag/js`)
